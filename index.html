Projet SmartCity Web Services Platform

Votre fil rouge : Développez une plateforme de Web Services pour une ville intelligente - REST, SOAP et intégration

🎯 Objectif du projet

Conception et développement d'une plateforme de Web Services pour une ville intelligente en utilisant les technologies REST et SOAP. Vous travaillerez en équipes de 3-4 étudiants avec une progression guidée étape par étape pour maîtriser tous les aspects des Web Services.

🏗️ Architecture cible

Services REST & SOAP
Sécurité OAuth 2.0 & JWT
Intégration WSDL/UDDI
Architecture SOA

📦 Services à développer

Events Service (REST)

Gestion des événements avec API REST

Transport Service (REST)

Informations transport avec pagination

Weather Service (SOAP)

Données météo via service SOAP

Auth Service (REST)

Authentification JWT + OAuth 2.0

📋 Progression Guidée

Chaque après-midi, vous implémenterez une partie spécifique du projet avec des instructions détaillées. Vous maîtriserez progressivement REST, SOAP, WSDL, UDDI et l'intégration de services.
