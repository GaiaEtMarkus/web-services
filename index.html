Projet SmartCity API Platform

Votre fil rouge : Développez une plateforme d'APIs REST pour une ville intelligente - Version simplifiée et réalisable

🎯 Objectif du projet

Conception et développement d'une plateforme d'APIs REST pour une ville intelligente en utilisant une approche Agile moderne avec progression guidée. Vous travaillerez en équipes de 3-4 étudiants avec des rôles tournants (PO, Dev Lead, DevOps, QA) et une progression étape par étape.

🏗️ Architecture cible

Microservices REST
Sécurité OAuth 2.0
Observabilité complète
Déploiement automatisé

📦 Services à développer (Version Simplifiée)

Events API

Gestion des événements de la ville (CRUD simple)

Transport API

Informations transport avec pagination

Weather Service

Données météo via API externe

Auth Service

Authentification JWT + OAuth basique

📋 Progression Guidée

Chaque après-midi, vous implémenterez une partie spécifique du projet avec des instructions détaillées. Si vous suivez la progression à la lettre, vous aurez terminé le projet fil rouge à la fin du 8ème après-midi.
